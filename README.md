Vitamin Dispenser Project

This repository contains all software, firmware, hardware, and documentation for the vitamin dispenser system.

Repository Structure
- 'docs/' - Project documentation (setup, architecture, API)
- 'firmware/' - Embedded code for the dispenser device (Arduino)
- 'backend/' - Server-side application and API
- 'mobile-app/' - Mobile application code
- 'hardware/' - CAD, schematics, and hardware documentation
